◆ step_down()

bool step_down ( int & x,
int & y )

Случайно сдвинуть объект на один шаг вниз (влево/вправо/прямо).

Общая логика движения, используемая и для обычных врагов, и для босса: с вероятностью 1/5 сдвиг влево, 1/5 — вправо, иначе прямо вниз. Сдвиг по X ограничен границами поля.

Parameters
xКоордината X (изменяется)
yКоордината Y (изменяется, всегда увеличивается на 1)
Returns
true, если после движения объект достиг нижнего края поля
